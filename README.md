## Installation

```bash
git clone https://github.com/RajveerSingh1321/pathfinding-visualizer
cd Pathfinding-Visualizer
npm install
npm start
```

## Algorithm List

Weighted:

- Dijkstra's algorithm

- A\* search algorithm (Manhattan distance heuristic)

Unweighted:

- Breadth-first search

- Depth-first search
